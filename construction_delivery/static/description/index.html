In construction projects there are 2 groups of users ,first group requests a task delivery with the project and task required and block and unit no. the project consists of blocks and each block consist of units ,second group are the approvers who accept or reject the task delivry after inspection

TO Create a record rule from the UI

  • 1] search for the model name "construction_delivery"
  • go to the record rule menu
  • 1]choose the required group
  • then add up this domain [('users', 'in',users.ids)]
